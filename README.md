# Banking Fraud Detection & Risk Analytics
Power BI project for fraud detection, customer risk segmentation, fraud amount prediction and business insights.
## Dashboard Pages
- Overview
- Fraud Prediction
- Customer Risk Segmentation
- Fraud Amount Prediction
- Feature Importance & Business Insights

## Key Insights
- Receiver balance is the strongest fraud indicator.
- Transfer and Cash-Out transactions contribute the highest fraud amounts.
- High-risk customers require enhanced monitoring.
- Fraud prediction helps prioritize suspicious transactions.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
